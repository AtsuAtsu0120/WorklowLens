# Source Generator

## 概要

Roslyn Source Generatorを活用し、2つの機能を提供する:
1. **型安全Action API**: ユーザー定義のenum（`[WorkflowActions]`属性付き）からCategoryLoggerの拡張メソッドを自動生成する
2. **自動計測メソッド**: `[WorkflowLog]`属性を付けたメソッドに対し、MeasureScopeラッパーを自動生成する

## 背景・目的

- `Log(Category.Build, "compile")`のような文字列actionはtypoリスクがある。型安全APIを生成することでIntelliSenseによる補完とコンパイル時検証を提供する
- MeasureScopeのusingボイラープレートを属性ベースで自動化し、コードの見通しを良くする

## 要件

### プロジェクト構成

- [ ] `src/WorkflowLensClient.Generators/`プロジェクトを新設する（netstandard2.0）
- [ ] `ISourceGenerator`（V1 API）を使用する（Unity 2021.3+互換）
- [ ] `Microsoft.CodeAnalysis.CSharp` 4.0.1を使用する（Unity同梱Roslynとの互換性）
- [ ] メインライブラリの`.csproj`からGeneratorを`OutputItemType="Analyzer"`で参照する

### 属性定義

- [ ] `WorkflowActionsAttribute` — enumに付与。対象Categoryを指定する
- [ ] `WorkflowLogAttribute` — メソッドに付与。Category, action, 公開メソッド名を指定する
- [ ] `WorkflowLensLoggerAttribute` — フィールドに付与。Generatorにloggerフィールドを通知する
- [ ] 属性クラスはGeneratorからソースとしても出力する（ユーザーが手動で参照を追加する必要をなくす）

### 型安全Action API（CategoryApiGenerator）

- [ ] `[WorkflowActions(Category.Build)]`が付いたenumの各メンバーに対し、CategoryLoggerの拡張メソッドを生成する
- [ ] 各enumメンバーに対して`Log`メソッドと`Measure`メソッドの2つを生成する
- [ ] enumメンバー名はPascalCase → snake_caseに変換してaction文字列とする
- [ ] 生成コードはC# 8.0準拠とする

### 自動計測メソッド（WorkflowLogGenerator）

- [ ] `[WorkflowLog]`属性が付いた`private`メソッドに対し、MeasureScopeで囲むpublicラッパーメソッドを生成する
- [ ] 含むクラスが`partial`であることを要求する
- [ ] `[WorkflowLensLogger]`属性が付いたフィールドをloggerインスタンスとして使用する
- [ ] 命名規約: `XxxCore`メソッド → `Xxx`公開メソッドを生成（属性パラメータでオーバーライド可能）
- [ ] C# 8.0の制約で拡張partial method（C# 9.0）は使えないため、Core/ラッパー方式を採用する
- [ ] 元メソッドのパラメータをそのままラッパーに引き継ぐ
- [ ] 元メソッドの戻り値がvoidでない場合も対応する

### 属性定義

```csharp
// Generatorがソースとして出力する属性
[AttributeUsage(AttributeTargets.Enum)]
public sealed class WorkflowActionsAttribute : Attribute
{
    public Category Category { get; }
    public WorkflowActionsAttribute(Category category) => Category = category;
}

[AttributeUsage(AttributeTargets.Method)]
public sealed class WorkflowLogAttribute : Attribute
{
    public Category Category { get; }
    public string Action { get; }
    public string? MethodName { get; set; }  // 省略時はCoreサフィックスを除去
    public WorkflowLogAttribute(Category category, string action) { ... }
}

[AttributeUsage(AttributeTargets.Field)]
public sealed class WorkflowLensLoggerAttribute : Attribute { }
```

### 型安全Action API — 生成例

ユーザー定義:
```csharp
[WorkflowActions(Category.Build)]
public enum BuildAction
{
    Compile,
    Link,
    ShaderCompile,
}
```

生成コード:
```csharp
// <auto-generated />
#nullable enable
using WorkflowLensClient;

public static class WorkflowLensBuildExtensions
{
    /// <summary>Build/compile ログを送信する</summary>
    public static void Compile(this CategoryLogger logger, long? durationMs = null)
        => logger.Log("compile", durationMs);

    /// <summary>Build/link ログを送信する</summary>
    public static void Link(this CategoryLogger logger, long? durationMs = null)
        => logger.Log("link", durationMs);

    /// <summary>Build/shader_compile ログを送信する</summary>
    public static void ShaderCompile(this CategoryLogger logger, long? durationMs = null)
        => logger.Log("shader_compile", durationMs);
}
```

使用例:
```csharp
// 自動計測 + 型安全action
using (var build = logger.Build("compile"))
{
    RunCompiler();
}

// グルーピング内で型安全action
using (var build = logger.Build())
{
    build.Compile(durationMs: 3200);
    build.Link();
}
```

### 自動計測メソッド — 生成例

ユーザーコード:
```csharp
public partial class MyToolController
{
    [WorkflowLensLogger]
    private readonly WorkflowLens _logger;

    [WorkflowLog(Category.Build, "compile")]
    private void CompileCore()
    {
        RunShaderCompiler();
    }

    [WorkflowLog(Category.Edit, "import", MethodName = "ImportAsset")]
    private AssetResult ImportAssetCore(string path)
    {
        return DoImport(path);
    }
}
```

生成コード:
```csharp
// <auto-generated />
#nullable enable
using System;
using WorkflowLensClient;

public partial class MyToolController
{
    public void Compile()
    {
        using (_logger.Build("compile"))
        {
            CompileCore();
        }
    }

    public AssetResult ImportAsset(string path)
    {
        using (_logger.Edit("import"))
        {
            return ImportAssetCore(path);
        }
    }
}
```

### PascalCase → snake_case変換

| 入力 | 出力 |
|------|------|
| `Compile` | `compile` |
| `ShaderCompile` | `shader_compile` |
| `BrushApply` | `brush_apply` |
| `LODGenerate` | `lod_generate` |

ルール: 大文字の前にアンダースコアを挿入し、全体を小文字化。連続大文字（略語）は最後の1文字を次の単語の先頭として扱う。

### エラーハンドリング（コンパイル時診断）

| コード | メッセージ | 条件 |
|--------|----------|------|
| `WLSG0001` | `[WorkflowActions]を付けたenumはpublicである必要があります` | enum が非public |
| `WLSG0002` | `[WorkflowLog]を使用するクラスはpartialである必要があります` | クラスが非partial |
| `WLSG0003` | `[WorkflowLensLogger]が付いたフィールドが見つかりません` | loggerフィールド未指定 |
| `WLSG0004` | `[WorkflowLog]メソッド名が"Core"で終わっていません` | 命名規約違反（Warning） |

### Unity向けパッケージング

1. Generator DLLをビルドし、Unityパッケージの適切なディレクトリに配置
2. `.dll.meta`で`RoslynAnalyzer`ラベルを付与（Unity 2021.3+）
3. `Microsoft.CodeAnalysis`はUnityコンパイラに同梱されているため、パッケージには含めない

## テスト方針

- [ ] CategoryApiGenerator: enumメンバーから正しい拡張メソッドが生成されること
- [ ] CategoryApiGenerator: PascalCase→snake_case変換が正しいこと（通常ケース、略語ケース）
- [ ] CategoryApiGenerator: 非publicなenumに対して診断WLSG0001が出ること
- [ ] WorkflowLogGenerator: Core/ラッパーメソッドが正しく生成されること
- [ ] WorkflowLogGenerator: 戻り値ありメソッドの対応
- [ ] WorkflowLogGenerator: パラメータ付きメソッドの引き継ぎ
- [ ] WorkflowLogGenerator: 非partialクラスで診断WLSG0002が出ること
- [ ] WorkflowLogGenerator: loggerフィールド未指定で診断WLSG0003が出ること
- [ ] テストは`CSharpGeneratorDriver`を使用して生成コードを検証する
